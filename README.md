# ArcDeck PixFlow

ArcDeck PixFlow é uma demo profissional de **cobrança usando Pix (simulado)** com **recibo público verificável onchain**, construída para a **Arc Testnet**.

> Aviso: “Pix” é marca registrada do Banco Central do Brasil. Este projeto não processa pagamentos reais — é uma simulação para testnet.

## O que tem aqui

- `web/` — Frontend (React + TypeScript + Tailwind + Framer Motion)
- `contracts/` — Contratos (Solidity) para a demo
- `docs/` — Documentação do produto, FAQ, termos e privacidade

## Como rodar o frontend

```bash
cd web
npm install
npm run dev
```

## Endereços (Arc Testnet)

- aBRL: ver `web/src/config/contracts.ts`
- Invoices: ver `web/src/config/contracts.ts`
