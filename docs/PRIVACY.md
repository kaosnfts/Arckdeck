# Política de Privacidade — ArcDeck PixFlow

## Dados coletados
- Endereço de carteira (necessário para operar o dApp)
- Preferência de tema (claro/escuro), armazenada localmente no navegador

## O que não coletamos
- Dados pessoais sensíveis
- Dados bancários
- Chaves privadas (nunca solicitadas)

## Armazenamento local
- `localStorage` é usado apenas para salvar o tema.

## Transparência
Eventos onchain são públicos por natureza. O aplicativo apenas lê e exibe dados.
